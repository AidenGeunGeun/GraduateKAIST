## 1. Requirements Structure

KAIST graduation has two layers:
1. **공통 이수요건** (Common) — applies to ALL students regardless of major
2. **학과별 전공과목 이수요건** (Department-specific) — varies by department

Requirements vary by **admission year (입학년도)**. The student's admission year locks in which version applies.

## 2. Common Requirements (2018이후 공통, applies to 2022학번)

### 2.1 기초필수 (Basic Required): 23학점

9 groups — pick one course from each:

| # | Options | Credits |
|---|---------|---------|
| 1 | 기초물리학I / 일반물리학I / 고급물리학I | 3 |
| 2 | 기초물리학II / 일반물리학II / 고급물리학II | 3 |
| 3 | 일반물리학실험I (required) | 1 |
| 4 | 기초생물학 / 일반생물학 | 3 |
| 5 | 미적분학I / 고급미적분학I | 3 |
| 6 | 미적분학II / 고급미적분학II | 3 |
| 7 | 기초화학 / 일반화학I / 고급화학 | 3 |
| 8 | 일반화학실험I / 고급화학실험 | 1 |
| 9 | 프로그래밍기초 / 고급프로그래밍 | 3 |

Total: 23학점

### 2.2 기초선택 (Basic Elective): 9학점 이상

Department-specific designated courses must be included. For AE: MAS109, MAS201, MAS202 중 2과목 이상 포함하여 9학점.

### 2.3 교양필수 (Liberal Arts Required): 7학점 + 8AU

**Credits (7학점):**
- English (4학점): EP&D(1) + Advanced Listening(1) + Advanced Writing(1) + Advanced Reading(1)
- 논술 (3학점)

**AU (8AU):**
- 체육과목: 4AU
- 인성/리더십: 2AU (= 32시간, e.g., 인성리더십I + II)
- 즐거운 대학생활: 1AU
- 신나는 대학생활: 1AU

### 2.4 인문사회선택 (HSS Elective): 21학점

3 계열: 인문, 사회, 문학과예술

**Distribution requirement**: 2계열 이상에서 각각 1과목씩 이상 (총 6학점 이상) 포함하여 21학점

복수전공 이수자: 계열 구분 없이 12학점 이상 (relaxed)

### 2.5 연구 (Research): 3학점 이상

- 졸업연구 3학점 필수
- AE students: can replace with AE401 항공우주시스템설계II
- 개별연구: max 4학점 as 연구학점

### 2.6 전공트랙 (Must complete one)

One of: 심화전공 / 부전공 / 복수전공 / 자유융합전공

## JSON Config Structure (for implementation)

Requirements should be stored as structured data. Here's the schema:

```json
{
  "admissionYearRange": [2016, 2022],
  "department": "AE",
  "totalCredits": 136,
  "categories": {
    "기초필수": { "required": 23, "courses": ["..."] },
    "기초선택": { "required": 9, "notes": "MAS109/201/202 중 2과목 포함" },
    "교양필수": { "requiredCredits": 7, "requiredAU": 8 },
    "인문사회선택": { "required": 21, "distribution": "2계열 이상" },
    "전공필수": { "required": 21, "courses": ["AE210","AE220","AE208","AE307","AE300","AE330","AE400"] },
    "전공선택": { "required": 21 },
    "연구": { "required": 3 }
  },
  "totalMajor": 42,
  "substitutions": {
    "AE210": ["ME211"],
    "AE230": ["ME231"],
    "AE311": ["ME311"],
    "AE370": ["ME301"]
  }
}
```

This structure makes it easy to add new departments or admission year ranges as additional config objects.
